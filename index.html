Share Dashboard with a Client

Host this HTML file on any static hosting (GitHub Pages, Netlify, Vercel — all free). Share the URL. Clients sign in with their Google account; as long as the connected Sheet grants them access, they'll see live data.

# Quick deploy steps:
1. Upload ZDJ_TradeVesting_Dashboard.html to GitHub repo
2. Enable GitHub Pages (Settings → Pages → Deploy from main)
3. Share URL: https://your-username.github.io/your-repo/
4. Configure Google OAuth at console.cloud.google.com
   → Add your Page URL to "Authorized JavaScript origins"
